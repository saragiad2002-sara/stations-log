
# PS Readings V02 — شهر تجريبي كامل

نسخة تجريبية كاملة تسمح بمراجعة/إدخال ملاحظات لكل **يوم تشغيلي** ضمن **شهر واحد**، مع:
- اليوم التشغيلي: 07:00 → 06:00
- تنبيه مجمّع قبل كل ساعة بـ 5 دقائق
- شاشة اختيار أيام الشهر + شاشة تفاصيل اليوم (ساعات 07→06)
- زر "ملاحظة" لكل موقع (PS1..PS7) داخل كل ساعة
- تخزين محلي Room + تحضير للمزامنة اللاحقة

## تشغيل وبناء APK
- افتح بالمحرّر Android Studio → Build APK(s) → `app-debug.apk`
- أو سطر أوامر: `./gradlew assembleDebug`
